Electrical Subsystem

Circuit Boards Built to Create More Circuit Boards

Electrical


The electrical subsystem has two main parts: the real time control system and the user interface system. The real time control system is based on an Arduino Uno V3. The user interface system is a Rasberry Pi B+ with a LCDTFT touchscreen. The two systems via serial through a USB cable.

User Interface System

A Raspberry Pi B+ commanding a Thin-Film-Transistor Liquid-Crystal Display touchscreen, provides the user with a visual interface with which to execute the printing process and abtracts away It also communicates with Arduino over Serial to transmit gcode commands and exchange information on operation status.

Real Time Control System

We chose to use an Arduino as our main real-time controller because it is the fast option to achieve a viable, albeit limited, product. Instead of purchasing a pre-designed Arduino shield which would limit our freedom while writing firmware, we (ironically) decided to create a perforated protoboard that easily sits on top of an Arduino Uno similar to other Arduino shields. We also created two extension board which connect to the main board for increased functionality and modualirty. By doing so, we were able to customize our electronics easily and at the same time leave room for future expansions as needed.

On the main board which sits directly on Arduino, we have three A4988 stepper motor drivers (one for each axis) and connections to limit switches and the stepper motors themselves.

On the the first extension board sits a couple of LEDs and a quadrature encoder switch. A press down on the quadrature encoder switch allows the user to cycle through between four modes with which to zero the axes prior to starting to mill: Manually (motor drivers disengaged), X, Y, and Z. The quadrature encoder ticks provides information on the distance and the direction the motor should be traveling in.

The second extension board has a power relay and a N-channel MOSFET to drive the z-axis spindle motor throughout the milling process. The power relay is separately controlled by the Arduino, ensuring the safe physical disconnect from a dangerous cutting tool from power when it is not enabled.

An E-stop switch can also be seen on the outer housing of the gantry for users to cut off power quickly in case of emergencies.

Power Management

The power distribution of our system is spilt between between the main control system and the z-axis actuation. Both are connected to a 12V wall adapter. The 12V for the control system is stepped down to 5V, which then goes onto power both Arduino and Raspberry Pi.

Thermal management is another important part of the electrical system. Due to the large amount of current that flow through the system to power the DC motor and three different stepper motors. Each stepper motor draws about 0.5A while the DC motor would pull about 1.3A. For passive heat dissipation, heat sinks are put on all three A4988 stepper motor modules while a large, through-hole N-MOSFET with a large heat conductive back is used. We also used a computer fan to circulate air actively across the electronics.
